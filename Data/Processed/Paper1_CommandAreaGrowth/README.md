# Paper 1 Processed Data

This folder contains intermediate and final analytical artifacts for the
command-area growth paper. These files are generated by scripts in:

`Code/paper1_command_area_growth/`

Manuscript-facing versions are compiled under:

`Output/Paper1_CommandAreaGrowth/`

## Subdirectories

| Directory | Contents |
| --- | --- |
| `yearly_command_areas/` | Active, overlap-free command-area layers for each AEI year. |
| `extracted_irrigation/` | Raster extraction diagnostics and intermediate inside/outside outputs. |
| `final_tables/` | Stable analytical CSVs used to build paper figures and tables. |

## Important Final Tables

| File | Purpose |
| --- | --- |
| `yearly_command_area_inventory.csv` | Command-area dam counts and footprints by year. |
| `inside_outside_irrigation_by_country_year_area_weighted.csv` | Primary inside/outside AEI panel. |
| `inside_outside_irrigation_by_country_year_all_touched.csv` | Extraction sensitivity panel. |
| `inside_outside_irrigation_by_country_year_center_cell.csv` | Extraction sensitivity panel. |
| `growth_decomposition_by_country_area_weighted.csv` | Primary 2000-2015 growth decomposition by country. |
| `growth_decomposition_summary_area_weighted.csv` | Primary regional growth summary. |
| `paper1_extraction_sensitivity_summary.csv` | Method sensitivity summary. |
| `paper1_command_area_dem_plausibility_summary.csv` | DEM QA summary. |
| `paper1_command_area_dem_plausibility_by_dam.csv` | Dam-level DEM QA. |
| `paper1_missing_dam_growth_rate_accounting.csv` | Missing-dam growth-rate context. |
| `paper1_missing_dam_tipping_points.csv` | Missing-dam tipping-point bounds. |
| `paper1_country_growth_rate_comparison.csv` | Country-level growth-rate diagnostics. |
