# Legacy Command-Area Notebooks

These notebooks were moved from `Code/Archive` so the paper branch keeps all exploratory command-area work close to the final workflow.

They are preserved for reference and should not be deleted. Before citing a result from one of these notebooks, port the relevant logic into `../notebooks/`, rerun it with current inputs, and save a stable output table or figure.

## Notebook guide

- `1_GDW_AEI_Targeting_Ratio.ipynb`: early distance-based dam/AEI targeting ratios.
- `2_Dam_Dataset_Comparison.ipynb`: dam dataset comparison work.
- `3_CA_Targeting_Ratio copy.ipynb`: CPIS-in-command-area targeting ratio prototype.
- `4_CA_Map_Figures_Fixed.ipynb`: map/alignment diagnostics for command areas and AEI rasters.
- `6_AEI_Ratio.ipynb`: country/year proportions of AEI and CPIS in command areas.
- `7_Prep_Descriptive_Stats.ipynb`: preparation of merged yearly command areas and masked irrigation rasters.
- `8_Descriptive_Stats.ipynb`: inside/outside pixel-level descriptive and distributional statistics.
- `9_Spatial_Statistics.ipynb`: exploratory spatial autocorrelation, hotspot, and fragmentation analyses.
- `10_CP_Descriptive_Stats.ipynb`: CPIS descriptive stats stub/reference.
- `reservoirs_without_dams.py`: utility script for reservoir/dam matching checks.
