# Paper 1 Notebooks

This directory is reserved for cleaned, paper-facing notebooks. The canonical
computations currently live one folder up as scripts.

Notebook work here should either:

- call the script-first workflow;
- read stable outputs from `Data/Processed/Paper1_CommandAreaGrowth/final_tables/`;
- or produce presentation/manuscript exploration without changing the core
  result.

Do not reimplement hidden analytical logic in a notebook if that logic affects a
paper result. Move it into the numbered scripts first.
